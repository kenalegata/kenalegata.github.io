Generator-FA

A Yeoman generator to kick-start your Front End web projects using Foundation 5, Assemble, SASS, SMACSS, BEM and Modernizr.

Getting Started

  • Install yo : npm install -g yo
  • Install SASS : gem install SASS (You need Ruby to do this.)
  • Install the generator : npm install -g generator-fa
  • Run yo fa to start your project
  • Run grunt server
